## Round Intent

R247 accepted the bounded wording sync that clarified:

- managed local loops and recursion route to `PROP-039+ or later`;
- service-loop progression source binding belongs to PROP-037 companion
  wording;
- `clock.every` is a progression `source_kind` / source binding, not
  semantically equivalent to `Stream[DateTime]`;
- `tick.time` is explicit event-time binding;
- source-level `now()` remains prohibited through Ch8 `OOF-L6`;
- `break` remains deferred;
- lab behavior remains frontier evidence only.

R248 tests whether that wording is ready for proof-local fixtures. The round
must not accidentally turn fixtures into implementation, runtime support, public
authority, or canonical acceptance of lab behavior.

This round must preserve:

- no live implementation authorization unless C1-A explicitly opens only the
  bounded proof fixture route;
- no `igc run` widening;
- no `.igbin` execution;
- no compiler passport emission;
- no RuntimeSmoke productization;
- no public runtime support;
- no Reference Runtime support;
- no stable API, production, Spark, release, public demo, public performance,
  official/reference status, alternative certification, or portability claims;
- no lab behavior accepted as canon.
