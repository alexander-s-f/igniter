# S3-R254 - Contract Invocation Forms SemanticIR Lowering

Status: Planned
Date: 2026-06-05

## IDD Classification

- Round type: authorization review + proof-local implementation + pressure
  review + acceptance decision + status curation
- Main route: `S3-R254-C1-A`
- Main track: `contract-invocation-forms-semanticir-lowering-design-authorization-review-v0`
- Purpose: decide whether bounded proof-local contract invocation forms
  SemanticIR lowering may begin after accepted R252 type-directed dispatch
  evidence and R253 route-resolution.

## Round Intent

R252 accepted proof-local contract invocation forms type-directed dispatch
evidence. R253 resolved the route-number collision and assigned this forms lane
to `S3-R254-C1-A`.

R254 should test whether resolved invocation forms can be lowered into an
explicit SemanticIR call/contract-invocation shape before any VM/runtime
execution boundary. The round is allowed to authorize a bounded proof-local lab
compiler proof only if C1-A explicitly opens that proof boundary.

This round does not authorize live mainline parser, TypeChecker, SemanticIR,
runtime, API, CLI, package, stable grammar, public API, `.igapp` execution,
`.igbin` execution, VM linker/subroutine frames, compiler passport emission,
RuntimeSmoke productization, public runtime support, Reference Runtime support,
release evidence, public performance claims, official/reference status,
alternative certification, portability guarantees, or lab behavior as canon.

`S3-R255` remains reserved for the Igniter Lang repository split boundary.
