## Round Intent

R256 accepted the repository split dry-run file-map as evidence only and held
physical migration. R257 is the next bounded preparation round. It turns the
R256 blockers into an explicit prep plan and proof packet:

- refreshed tracked-file count;
- mutually exclusive destination dispositions;
- support-file copy/synthesize/rewrite/defer matrix;
- path/link hygiene handling plan;
- generated artifact exclude/quarantine/whitelist matrix;
- target repository baseline recommendation;
- migration-method prerequisites.

R257 does not authorize repository migration, target population, history
rewrite, `git subtree split`, `git filter-repo`, remote push, package rename,
package/CI changes, release execution, public claims, framework-to-language
authority transfer, lab canon, or archive repo creation.
