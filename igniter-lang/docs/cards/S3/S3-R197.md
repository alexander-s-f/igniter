Status: planned
Date: 2026-05-27

Dispatch: R197 = C1-A -> C2-I -> C3-X -> C4-A -> C5-S

- C1-A decides whether a proof-local `if_expr` runtime/evaluator experiment may
  begin.
- C2-I implements the proof-local evaluator experiment only if C1-A authorizes
  it.

  boundary.
- C4-A accepts, conditionally accepts, holds, or redirects the proof-local

- Prove lazy branch execution without touching live runtime/library surfaces.
- Preserve release/public/Spark/API/CLI and compiler behavior boundaries.
