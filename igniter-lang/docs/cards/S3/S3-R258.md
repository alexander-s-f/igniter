## Round Intent

R257 conditionally accepted support-file/path-hygiene prep as
migration-readiness evidence only and opened R258 as an authorization-review
gate. R258 must decide whether the split is ready for a later bounded execution
route, or whether additional file-map, path/link, support-file, generated
artifact, target-baseline, or package-metadata prep is still required.

R258 is not a live migration round. It does not authorize or run `git subtree
split`, `git filter-repo`, copy/populate target repositories, remote push,
release, package/CI changes, archive repo creation, public claims,
framework-to-language authority transfer, or lab canon.
