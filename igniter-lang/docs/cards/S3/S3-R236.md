Status: Planned
Date: 2026-06-02

## IDD Classification

Mode:

```text
standard / ecosystem pressure map + next-route decision
```

Why:

- R234 accepted bounded experimental `igc run` Slice 0 implementation as
  delegated-runtime evidence only;
- R235 landed bounded quickstart/docs exposure without opening public runtime,
  Reference Runtime, stable API, production, Spark, release, `.igbin`,
  RuntimeSmoke, or compiler-passport authority;

  multi-component lab ecosystem: Rust compiler, VM/runtime, stdlib, TBackend,
  app-consumer surfaces, and integration adapters;
- lab energy is useful, but mainline must translate it into explicit evidence
  classes and next-route choices before any authority can drift.

route decision

productization after Slice 0 docs, while classifying the alternative lab
ecosystem as pressure/evidence rather than authority.

execution, compiler passport emission, RuntimeSmoke productization, Reference
Runtime support, public runtime support, stable API, production readiness,
Spark integration, release execution, public docs claims, public performance
claims, Official Reference Implementation status, alternative certification, or
artifact portability guarantees.
