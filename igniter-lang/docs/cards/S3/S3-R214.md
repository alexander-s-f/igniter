runtime, report, API, or public-facing work is considered.


Use this as pressure, not as authorization. After the counterfactual layer is
consolidated, the Main Line should decide whether a runtime-debt and
time-to-market review opens next. That later review may inspect runtime debt,
but this R214 round must not authorize runtime implementation.

Do not authorize or perform:

- implementation;
- `lib/**` edits;
- parser, grammar, source syntax, TypeChecker, SemanticIR, runtime,
  evaluator, RuntimeSmoke, proof RuntimeMachine, CompilerOrchestrator,
  CompilerResult, or CompilationReport changes;
- report/result/receipt/CompatibilityReport shape changes;
- `.igapp` schema or golden artifact changes;
- live non-selected branch evaluation;
- dependency/cache authority changes;
- public docs, public demo, release, API/CLI, Spark, production, stable,
  all-grammar, or runtime support claims.
