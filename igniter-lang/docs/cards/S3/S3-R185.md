# S3-R185 - igniter_lang 0.1.0.alpha.1 Release Execution

Status: planned
Date: 2026-05-26

Dispatch: R185 = C0-U -> C1-I -> C2-X -> C3-A -> C4-S

Pattern:
- C0-U is a human approval packet, not an agent card.
- C1-I may execute only after C0-U approval is present and exact enough.
- C2-X runs only after C1-I produces an execution receipt or a HOLD/abort receipt.
- C3-A accepts, holds, or routes incident/docs follow-up after C2-X.
- C4-S curates final round status after C3-A.

Round intent:
- Execute the bounded RubyGems alpha prerelease for `igniter_lang 0.1.0.alpha.1`
  if and only if the accepted R184 boundary and explicit user approval are both
  satisfied.
- Preserve the R183 evidence chain by rebuilding the gem and requiring the
  rebuilt artifact SHA256 to match the accepted R183 SHA.
- Verify public RubyGems availability before pushing the exact tag.
- Do not widen claims or surfaces.

Global constraints:
- Do not publish without exact user approval.
- Do not publish if rebuilt SHA differs from the accepted R183 SHA.
- Do not run `rake release`.
- Do not run `git push --tags`.
- Do not publish any gem other than `igniter_lang-0.1.0.alpha.1.gem`.
- Do not yank automatically.
- Do not sign or deploy.
- Do not edit version, gemspec, README, RELEASE_NOTES, compiler, runtime, or
  Spark code in this round.
- Do not authorize stable, production, public demo, all-grammar, branch/
  conditional `if_expr`, profile finalization/discovery/defaulting, Spark,
  runtime, signing, or deployment claims.

does not authorize publish or tag push.
