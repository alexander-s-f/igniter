# Igniter-Lang Reports

Status: active
Owner: [Igniter-Lang Supervisor]
Date: 2026-05-20

Igniter-Lang may use existing `stage*-round*-status-curation` tracks as report
packets when they include:

- compact summary;
- completed cards;
- evidence;
- blockers;
- cross-lane requests;
- recommended next route.

If a status-curation track is insufficient for Portfolio review, add a compact
report packet here.

Core rule:

```text
No report packet or status-curation equivalent -> lane round is not closed for Portfolio.
```
