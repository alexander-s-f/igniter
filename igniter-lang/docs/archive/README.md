# Igniter-Lang Documentation Archive

This directory contains cold archive snapshots of the Igniter-Lang documentation.

## Purpose

Archive snapshots preserve full documentation state at significant project milestones.
They are **read-only archaeology context** — not working documents.

Use the archive when:
- Tracing the origin of a design decision
- Recovering research that was deferred from active docs
- Understanding what was known before a major phase transition

Do not use the archive as the working context. Use `igniter-lang/docs/` instead.
