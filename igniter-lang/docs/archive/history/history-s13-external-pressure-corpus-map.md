## Strong Values To Preserve

- **Accountable result over raw result:** outputs need evidence, time, authority,
  receipt, and replay context.
- **Boring app proof before spectacular demos:** HTTP/JSON, CRUD, Spark-style
  operations, and typed receipts are better first tests than robotics or
  self-hosting claims.
- **External boundaries are typed obligations:** external calls need effect
  class, capability, failure semantics, idempotency, and receipt rules.
- **Proposal before action:** human review, correction, acceptance, override,
  and MeaningDiff patterns recur across OSINT, medicine, generated artifacts,
  agent mesh, and operations.
- **Simulation is not fact:** real, synthetic, forecast, counterfactual, model,
  and human-review observations must not collapse into one evidence class.
- **Safety claims must be measured:** hard real-time, collision avoidance,
  clinical safety, legal correctness, and escrow cannot be promised by syntax.
- **Graph-native authoring matters:** users want named value graphs, not
  imperative `compute {}` as the default mental model.

## Superseded Or Not Canon

- Exact Agent-A syntax is not accepted grammar.
- Exact Agent-B syntax is not accepted grammar.
- Agent-A claims that Igniter-Lang is already general-purpose are not proof.
- `external pure` for HTTP, model calls, robot commands, money, storage,
  messaging, or hardware is unsafe and should not return as canon.
- Live hardware actuation, neuro stimulation, legal/medical autonomy,
  self-replication, real custody/escrow, and anti-censorship operational evasion
  are not product targets.
- `compute {}` as the primary ordinary authoring surface is not a preserved
  value; it is pressure for a better clean-form language.
- `Any` as a routine contract boundary is a type-safety leak, not a desired
  pattern.
- Tensor/autograd/operator overloading/self-hosting claims are long-horizon
  pressure, not near-term core commitments.
