# igniter-ai

Local monorepo gem that owns Igniter's AI capability pack:

- `Igniter::AI`
- AI executors and skills
- tool registry and tool-use loop support
- LLM providers
- transcription providers and result types

Primary entrypoints:

- `require "igniter-ai"`
- `require "igniter/ai"`
