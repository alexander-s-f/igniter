# igniter-agents

Local monorepo gem that owns Igniter's actor runtime and agent libraries:

- `Igniter::Agent`, `Igniter::Registry`, `Igniter::Supervisor`
- `Igniter::Agents`
- `Igniter::AI::Agents`

Primary entrypoints:

- `require "igniter-agents"`
- `require "igniter/agent"`
- `require "igniter/supervisor"`
- `require "igniter/registry"`
- `require "igniter/agents"`
- `require "igniter/ai/agents"`
