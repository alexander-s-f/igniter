# igniter-server

Local monorepo gem for Igniter's built-in HTTP server transport layer.

Clean contracts-native application host adapter:

- `Igniter::Server::ApplicationHost`
- `Igniter::Server::ApplicationConfigProjection`

This adapter is intended for the new `igniter-application` package and should
be preferred over legacy `igniter-app` hosting for new work.
