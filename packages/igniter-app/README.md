# igniter-app

Local monorepo gem that owns Igniter's application runtime layer:

- `Igniter::App`
- app hosts, loaders, schedulers, diagnostics, evolution, and generators

Primary entrypoints:

- `require "igniter-app"`
- `require "igniter/app"`
- `require "igniter/app/runtime"`
- `require "igniter/app/scaffold_pack"`
