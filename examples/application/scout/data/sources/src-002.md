## Summary

Mid-size engineering teams value quick feedback loops and local autonomy when
adopting AI coding assistants.

## Claims

- Teams reported faster first drafts when assistants were available in the IDE. [p1]
- Developers valued explanation and test suggestions more than full code generation. [p2]
