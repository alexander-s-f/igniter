## Proposal

Move user profile storage from PostgreSQL to MongoDB so product teams can add
profile attributes without database migrations.

## Rationale

The current relational schema slows experimentation. A document store looks
attractive for flexible profile fields, but the proposal needs to be checked
against previous data and PII decisions before implementation starts.
